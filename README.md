## 预览地址  [https://kaife.cn/](https://kaife.cn/)

## Usage

拉取代码

```sh
git clone https://github.com/xkfe/blog.git
```

安装依赖
```sh
cd blog
pnpm install
```

开发启动
```sh
pnpm dev
```

构建
```sh
pnpm build
```

预览产物
```sh
pnpm serve
```
